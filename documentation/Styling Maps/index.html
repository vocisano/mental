Styling Maps

How do I style a map as a whole?

Tap the Paintbrush button in the toolbar. The style menu appears, that allows you to change style settings for concepts, relationships, and connectors.

Style options include :

  • Font face,
  • Text size,
  • Text color,
  • Fill color,
  • Stroke color,
  • Stroke width.

How do I style a single element or a group of elements?

Select the elements you wish to style. Tap the Paintbrush button in the toolbar.
